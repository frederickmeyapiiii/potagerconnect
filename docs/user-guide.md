# Guide utilisateur

## Objectif
PotagerConnect permet de suivre les jardins partagés, de consulter les parcelles, de gérer les tâches, de voir les photos et de publier dans le forum.

## Navigation
- `Parcelles` : section de suivi des parcelles et des jardiniers.
- `Planning` : section des tâches à réaliser selon les saisons.
- `Forum` : espace pour publier un message, partager des conseils et proposer des échanges.
- `Voir les récoltes` : carrousel photo pour visualiser l’évolution des cultures.

## Utilisation
1. Ouvrir l’application à l’adresse `http://localhost:5000`.
2. Consulter les statistiques en haut de la page.
3. Rechercher les parcelles et leur statut dans la section `Cartographie du jardin`.
4. Ajouter une parcelle avec le formulaire dans la section `Cartographie des parcelles`.
5. Ajouter une tâche dans la section `Planning de plantation`.
6. Publier un message dans le forum en remplissant le formulaire et en cliquant sur `Publier`.
7. Cliquer sur une image dans la galerie pour voir les détails et le suivi.

## Actions disponibles
- Ajouter un message dans le forum.
- Consulter les photos d’évolution des cultures.
- Voir les statuts de chaque parcelle.
- Ajouter une nouvelle parcelle ou une nouvelle tâche.
